# 🏥 Health Trend Dashboard V3

네이버 자동완성 API로 120개 씨앗 키워드의 연관 검색어를 **매일 자동 수집**하고,
사람들이 실제로 무엇을 검색하는지 트렌드를 분석하는 대시보드입니다.

---

## 프로젝트 구조

```
health-trend-report-v3/
├── .github/
│   └── workflows/
│       └── collect.yml       # GitHub Actions 자동화 (매일 오전 9시 KST)
├── data/                     # 수집된 CSV 저장 폴더
│   ├── top20_YYYY-MM-DD.csv  # 날짜별 TOP 20 키워드
│   └── related_YYYY-MM-DD.csv# 씨앗 키워드별 연관 검색어
├── src/
│   └── collect.py            # 네이버 자동완성 수집 스크립트
├── dashboard.py              # Streamlit 대시보드
├── requirements.txt
└── README.md
```

## 사용 방법

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 데이터 수집 (수동)
```bash
python src/collect.py
```

### 3. 대시보드 실행
```bash
streamlit run dashboard.py
```

---

## GitHub Actions 자동화

`.github/workflows/collect.yml`이 매일 **오전 9시 KST(00:00 UTC)**에 자동 실행됩니다.

- 수집 결과를 `data/` 폴더에 CSV로 저장
- 자동으로 커밋 & 푸시
- GitHub Actions 탭에서 **수동 실행**도 가능 (`workflow_dispatch`)

---

## 대시보드 기능

| 탭 | 기능 |
|----|------|
| 📊 TOP 20 트렌드 | 날짜별 가장 많이 등장한 연관 검색어 TOP 20 차트 + 순위표 |
| 🔍 씨앗 키워드별 연관어 | 씨앗 키워드 선택 → 자동완성 연관어 목록 + 수집 현황 |
| 📈 키워드 시계열 | 여러 날짜에 걸친 키워드 등장 횟수 추이 + 히트맵 |
